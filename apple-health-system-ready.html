🚀 How to Use the System

1 Access VitalSense: Open http://localhost:5173 in your browser
2 Navigate to Data Upload: Click on the "Live Health Data System" section
3 Choose Upload Method:
  • Apple Health: Export from iOS Health app, extract export.xml, and upload
  • JSON/CSV: Upload structured health data files
  • Manual Entry: Input individual metrics manually
  • Live Simulation: Generate realistic test data
4 View Analysis: Navigate through Gait Analysis, Walking Patterns, and Posture Analysis for comprehensive health insights

🔍 Apple Health Export Process

To get your Apple Health data:

iPhone Health App → Profile → Export All Health Data → Share/Save ↓ apple_health_export.zip ├── export.xml ← Upload this file ├── workout-routes/ └── electrocardiograms/

Note: Currently, ZIP files require manual extraction. Upload the export.xml file directly for processing.
